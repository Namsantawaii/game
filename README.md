# 타이어트랙 양재점 웹사이트 🏁

슈퍼카 전문 타이어 정비소 '타이어트랙 양재점'의 프리미엄 웹사이트입니다.

## 🎯 프로젝트 개요

블랙노드 스타일의 현대적이고 세련된 디자인으로 제작된 타이어 전문점 웹사이트입니다.
3D 휠얼라인먼트와 슈퍼카 전문 작업을 제공하는 프리미엄 서비스를 효과적으로 전달합니다.

## 🛠 기술 스택

- **Next.js 14** (App Router) - 최신 React 프레임워크
- **TypeScript** - 타입 안정성
- **Tailwind CSS** - 유틸리티 기반 스타일링
- **Lucide React** - 현대적인 아이콘
- **Shadcn UI** - 고품질 UI 컴포넌트

## 🎨 디자인 테마

- **Primary Color**: Deep Navy (#1e3a8a) - 신뢰감과 전문성
- **Secondary Color**: Racing Red (#dc2626) - 역동성과 열정
- **반응형 디자인**: 모바일/태블릿/데스크톱 최적화
- **프리미엄 UX**: 그라데이션, 블러 효과, 부드러운 애니메이션

## 시작하기

### 설치

```bash
npm install
```

### 개발 서버 실행

```bash
npm run dev
```

브라우저에서 [http://localhost:3000](http://localhost:3000)을 열어 확인하세요.

### 빌드

```bash
npm run build
```

### 프로덕션 실행

```bash
npm start
```

## 📁 프로젝트 구조

```
game/
├── app/
│   ├── page.tsx              # 메인 랜딩 페이지 (히어로, 서비스, 장비, 신뢰, 프로세스, CTA)
│   ├── pricing/page.tsx      # 타이어 가격 페이지
│   ├── booking/page.tsx      # 예약하기 페이지
│   ├── location/page.tsx     # 오시는 길 페이지
│   ├── layout.tsx            # 루트 레이아웃 (헤더/푸터 포함)
│   └── globals.css           # 글로벌 스타일 (커스텀 컬러 변수)
├── components/
│   ├── SiteHeader.tsx        # 네비게이션 헤더 (데스크톱/모바일)
│   ├── SiteFooter.tsx        # 프리미엄 푸터
│   └── ui/                   # Shadcn UI 컴포넌트들
├── lib/
│   └── utils.ts              # 유틸리티 함수
└── tailwind.config.ts        # Tailwind 설정 (커스텀 테마)
```

## ✨ 주요 기능

### 랜딩 페이지
- 🎯 **히어로 섹션**: 그라데이션 배경, 실시간 매장 정보 카드
- 🔧 **서비스 소개**: 프리미엄 타이어 교체, 3D 휠얼라인먼트, 긴급 출동
- ⚙️ **전문 장비**: 3D 고속 휠발란서, 슈퍼카 전용 리프트 등
- 🏆 **신뢰의 이유**: 10년+ 경력, 고객 만족도 98%, AS 보증
- 📋 **작업 프로세스**: 3단계 간편 프로세스
- 🚀 **CTA 섹션**: 예약 유도 및 오시는 길 안내

### UI/UX
- ✅ 반응형 네비게이션 (데스크톱 메뉴바 + 모바일 햄버거)
- ✅ 모바일 우선 반응형 디자인
- ✅ 부드러운 스크롤 애니메이션
- ✅ 접근성 고려 (ARIA 레이블, 키보드 네비게이션)
- ✅ SEO 최적화 (메타데이터, 시맨틱 HTML)
